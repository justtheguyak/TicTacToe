## ğŸ“– About The Project


This project is a visually engaging recreation of Tic Tac Toe built with vanilla web technologies. It moves away from the traditional pen-and-paper look, offering a glowing neon aesthetic, particle background effects, and fluid CSS animations.

It includes a robust state management system in JavaScript to handle game logic, win detection, and a CPU opponent that offers a challenge without being impossible to beat.

## ğŸš€ Getting Started

To get a local copy up and running, follow these simple steps.

### Prerequisites

You need a modern web browser (Chrome, Firefox, Safari, Edge). No Node.js or npm installation is required as this is a static web project.

### Installation

1.  **Clone the repository**
    ```sh
    git clone [https://github.com/justtheguyak/TicTacToe.git](https://github.com/justtheguyak/TicTacToe.git)
    ```
2.  **Navigate to the project directory**
    ```sh
    cd TicTacToe
    ```
3.  **Run the project**
    * Simply double-click `index.html` to open it in your browser.
    * OR, if you use VS Code, use the "Live Server" extension for the best experience.

## ğŸ•¹ï¸ How to Play

1.  **Choose your Mode:** Select "PvP Mode" for 2 players or "Play vs Bot" to play against the computer.
2.  **Make a Move:** Click on any empty cell to place your symbol (X).
3.  **Winning:** The first player to align 3 symbols horizontally, vertically, or diagonally wins.
4.  **Reset:** Use the "New Game" button to clear the board or "Reset Scores" to restart the counter.
